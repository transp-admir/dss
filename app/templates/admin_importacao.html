1. Importar Motoristas

Importe motoristas a partir de um arquivo .xlsx ou .csv.

Formato esperado:
  • Obrigatórias: nome, cpf, unidade
  • Opcionais: rg, cnh, frota, operacao
  • A 1ª linha deve conter os cabeçalhos.
  • Motoristas com CPF já cadastrado serão ignorados.
2. Importar Placas

Importe placas avulsas. As placas devem existir antes de montar os conjuntos.

Formato esperado:
  • Obrigatórias: numero, tipo, unidade
  • Opcional: operacao
  • O tipo deve ser CAVALO ou CARRETA.
  • Placas com número já cadastrado serão ignoradas.
3. Importar Conjuntos Formados

Monte os conjuntos (veículos) a partir das placas já importadas.

Formato esperado:
  • Obrigatórias: nome_conjunto, unidade, placa_cavalo
  • Opcionais: placa_carreta1, placa_carreta2, operacao, obs
  • Use o número das placas para preencher as colunas.
  • Conjuntos com nome já cadastrado serão ignorados.
